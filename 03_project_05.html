Overview

TOODO is a to-do list App which utilizes drag & drop gesture to generate a distinct user experience. Comparing with traditional to-do list App, TOODO has a clean UI that consists different icons. All user action such as adding, deleting and viewing can be done by drag and drop.

Implementation

The App UI is constructed using SVG elements. Canvas is also an option. But SVG is easier to work with and less conflict with other HTML elements.

User Expereince

To feature the drag and drop gesture, there are different drop zones in the App UI. Right and left are deleting zone, and Bottom is view and edit zone. This way we can provide a brand new user experience and fasten the task performance at the same time.

During the user test, some participants have confirmed that the App is more efficient and more fun to operate because icon provide a quick visual cognition. However, other participants have also commented that the App is lack of hint for new users. Since the icon seems clickable, but actually they cannot.

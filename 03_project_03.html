Concept

The application, "Drive Safe", is intended for new drivers who have finished the written test and are heading to the road test. We have considered that there is a huge gap between a written test and an actual driving experience. Understanding theory will not help completely as driving requires hands on experience. Thus, Drive Safe is a driving simulation that will bridge this gap between written theory and the real driving experience.

In this project, I was responsible for the system design including interaction between user input and system response, also the system flow based on the scenario.

Design Iteration

Besides the simulator, we have focused on the user learning experience. In order to teach users to perform each driving action correctly, the system provides feedback to notify any incorrect or missed steps like shoulder check or turning on signals.

At first, we used text notification. But some users reflected that text takes time to read which will disturb their driving performance.

Next, we changed the text to icon notification, which may be hard to be understood by new users, but the icon is fast to perceive and easier to remember.

Furthermore, we added prohibition signs to emphasize user's mistakes in order to give more impression and help users to avoid mistakes.
